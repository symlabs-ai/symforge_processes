# SymForge Processes

Repositório de templates de processos usados pelo app SymForge para orquestrar fluxos com symbiotas.

## O que tem aqui
- `docprocess/`: processo de criação e publicação de artigos (coleta de materiais, definição de estilo, escrita, validação, publicação em WordPress e notificação).
- `.gitignore`: mantém diretórios locais (ex.: `forgeprocess/`, `forgebase/`) fora do versionamento.

## Como usar
1) Consulte `docprocess/README.md` para entender o fluxo e os symbiotas envolvidos.  
2) Use `docprocess/process.yml` se precisar integrar o fluxo a automações do SymForge.  
3) Verifique `docprocess/VERSION.md` para a versão atual do pacote de processos.
