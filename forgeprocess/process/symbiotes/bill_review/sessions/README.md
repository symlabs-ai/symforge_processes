# Sessões do Bill Review

Este diretório armazena os **registros internos de raciocínio** do agente Bill Review.

## Propósito

Cada arquivo aqui representa uma sessão de análise e revisão conduzida pelo Bill durante o processo. Estes são registros **internos e privados** do agente, diferentemente das sessões formais do projeto.

## Formato

Arquivos devem seguir o padrão:
```
coach_YYYY-MM-DD_<descrição>.md
```

Exemplo:
- `bill_2025-10-17_review-feature-x.md`
- `bill_2025-10-17_analise-qualidade.md`

## Diferença entre Sessions

| Local | Tipo | Público |
|-------|------|---------|
| `symbiotas/bill_review/sessions/` | Interno | Agente |
| `project/docs/sessions/` | Formal/Oficial | Stakeholders |

Consulte o README.md principal para mais detalhes sobre o processo de revisão.
