# Sessões do Jorge Forge

Este diretório armazena os **registros internos de raciocínio** do agente Jorge Forge.

## Propósito

Cada arquivo aqui representa uma sessão de análise e revisão conduzida pelo Jorge durante o processo. Estes são registros **internos e privados** do agente, diferentemente das sessões formais do projeto.

## Formato

Arquivos devem seguir o padrão:
```
coach_YYYY-MM-DD_<descrição>.md
```

Exemplo:
- `jorge_2025-10-17_review-sprint-1.md`
- `jorge_2025-10-17_analise-feedback.md`

## Diferença entre Sessions

| Local | Tipo | Público |
|-------|------|---------|
| `symbiotas/jorge_forge/sessions/` | Interno | Agente |
| `project/docs/sessions/` | Formal/Oficial | Stakeholders |

Consulte o README.md principal para mais detalhes sobre o processo de revisão.
