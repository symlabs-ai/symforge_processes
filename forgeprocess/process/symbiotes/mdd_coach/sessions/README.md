# Sessões do MDD Coach

Este diretório armazena os **registros internos de raciocínio** do agente MDD Coach.

## Propósito

Cada arquivo aqui representa uma sessão de diálogo e análise conduzida pelo Coach durante o processo MDD. Estes são registros **internos e privados** do agente, diferentemente das sessões formais do projeto.

## Formato

Arquivos devem seguir o padrão:
```
coach_YYYY-MM-DD_<descrição>.md
```

Exemplo:
- `coach_2025-10-17_entrevista-inicial.md`
- `coach_2025-10-17_analise-hipotese.md`

## Diferença entre Sessions

| Local | Tipo | Público |
|-------|------|---------|
| `symbiotas/mdd_coach/sessions/` | Interno | Agente |
| `project/docs/sessions/` | Formal/Oficial | Stakeholders |

Consulte o README.md principal para mais detalhes sobre a arquitetura MDD.
